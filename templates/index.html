如何設定 LINE 機器人？

  1. 造訪 LINE Developers Console 並登入。
  2. 點擊 "Create a new provider" 建立服務商。
  3. 點擊 "Create a Messaging API channel" 建立機器人頻道。
  4. 在 Messaging API 設定頁籤中掃描 QR Code 將 Bot 加為好友。
  5. 在該頁籤底部點選 Issue 取得 Channel Access Token。
  6. 在 Basic Settings 頁籤最下方複製您的 User ID。
  7. 在左側表單中填入並點選儲存。
